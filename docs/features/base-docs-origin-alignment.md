# Base Docs Origin Alignment

## Resumo

Atualizacao documental para consolidar o chat original como origem historica do Orun.AI e alinhar os documentos base do projeto.

## Motivacao

O projeto nasceu em uma conversa longa que passou por:

- escolha do nome Orun.AI;
- identidade visual inspirada em Orun/Orunmila/Ifa;
- slogan e posicionamento institucional;
- pesquisa sobre agentes de IA, MCP e JavaScript;
- analise de fluxos comerciais;
- criacao do conceito Orun.AI Workforce Platform.

Os documentos base precisavam explicitar essa origem para evitar perda de contexto e manter coerencia em futuras solicitacoes.

## Arquivos Atualizados

- `README.md`
- `docs/prd.md`
- `docs/workflows.md`
- `docs/orchestrator.md`
- `AGENTS.md`
- `DESIGN.md`
- `CLAUDE.md`

## Decisoes

- O chat original foi usado como fonte historica bruta, consolidado nos documentos base e removido como arquivo separado para evitar duplicidade.
- MCP no projeto significa prioritariamente Model Context Protocol.
- Multi-component planning e tratado como capacidade arquitetural do Orchestrator, nao como substituto do protocolo.
- A inspiracao cultural da marca deve ser usada de forma respeitosa, institucional e nao religiosa.
- O foco inicial continua sendo automacao comercial, com arquitetura preparada para AI Workforce.
