# Plano: Validação das Correções de Invocação de Agentes

## Contexto do Erro

O erro ocorreu quando um agente chamou a ferramenta `actor` incorretamente:

```json
{
  "operation": "run_agent",    // ← STRING, deveria ser OBJECT
  "timeout_ms": 30000,          // ← Chave extras na raiz (fora do schema)
  "context": { "leadId": "123" } // ← Chave extras na raiz (fora do schema)
}
```

O schema correto exige que `operation` seja um objeto com as chaves internas `action`, `subagent_type`, `description`, `prompt`, etc. Campos como `timeout_ms` e `context` só existem DENTRO de `operation`, não na raiz.
